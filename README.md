# IRMA Configuration files

Stores configuration files per issuer/relying party. Typical directory structure:

	Organization
	+-- Issues
	|   +-- CredetialName
	|   	+--- issuingDescription.txt
	|   	+--- structure.xml
	+-- Verifies
	|   +-- VerifySpecName
	|   	+--- verifyDescription.txt
	|   	+--- specification.xml
	+-- description.txt
    +-- baseURL.txt
	+-- gp.xml
	+-- ipk.xml
	+-- sp.xml
